# 游戏实体形象设计详细文档

## 一、技术约定

所有实体使用 **Phaser.GameObjects.Graphics** 在 **局部原点 (0,0)** 绘制，绘制后通过 `gfx.setPosition(x, y)` 放置到场景中。

绘制函数签名：`drawXxx(gfx, s)`，其中 `s` 为缩放系数。

### 统一缩放（自2026-06-18更新）
所有游戏实体（怪物、精英、Boss、NPC、商人）统一使用 **1.5倍缩放**，与骑士玩家尺寸一致：

| 实体类型 | 原始缩放 | 当前缩放 |
|---------|---------|---------|
| 普通怪物（6种） | 1.0 | **1.5** |
| 精英怪物（5种） | 1.0 | **1.5** |
| Boss（5种） | 1.0 | **1.5** |
| 奇遇NPC（4种） | 1.4 | **1.5** |
| 商人 | 1.2 | **1.5** |

### 标签/指示器位置调整
由于实体放大，标签Y偏移同步增大：
- 怪物名称标签：`y + 24` → **`y + 34`**
- NPC名称标签：`npcY + 20` → **`npcY + 30`**
- NPC白色倒三角指示器偏移：`- radius - triSize - 4` → **`- radius - triSize - 14`**

### 命名匹配逻辑
- 名称包含特定中文/英文关键词 -> 选择对应绘制函数
- `mon.specialType === 'FLAME_SLIME'` 覆盖名称匹配

## 二、玩家（中世纪骑士）

### 结构
使用 `Phaser.GameObjects.Container`，包含两个子 Graphics：
- **bodyGfx**：身体部分（固定朝右，不旋转）
- **swordGfx**：剑部分（局部原点在护手处，`rotation` 随移动方向旋转）

### 技术参数
- 缩放：原始尺寸（s=1）
- 剑默认方向：水平向右
- 剑旋转：`scene.playerSwordGfx.rotation = scene.facingAngle`
- 剑位置偏移（相对容器）：`(13, -2)`（肩膀处）

### 绘制层次（从上到下）

**头盔**（中心 y=-13，半径6）：
1. 头盔主体圆形（灰蓝）
2. 面罩矩形（暗蓝，-3,-16 到 6×5）
3. 面罩横条T形呼吸口（金属镶边线条）
4. 头顶盔冠（深色矩形 + 金属圆球顶）

**护颈**（y=-8，7×3矩形，金属镶边）

**胸甲**（y=-5，16×12矩形）：
- 主体填充 + 深色描边
- 中线竖纹
- 上半部半透明高光条

**肩甲**（左右各6×5矩形，深色+镶边）

**腰带**（y=5，16×3矩形，金黄扣4×3）

**腿甲**（左右各6×8矩形）：
- 主体+深色描边
- 膝盖处圆形金属纹

**铁靴**（左右各6×4矩形，深色+镶边）

**盾牌**（左臂，固定）：
- 古铜主体6×10矩形+底部半圆
- 金色镶边描边
- 十字金属纹装饰

**右手臂**（固定铠甲袖，从肩膀伸出）

**剑**（swordGfx，从肩膀处延伸）：
- 铁手套（护手处小矩形）

- 剑柄（深棕线条向下）
- 剑柄圆头
- 剑刃（银白水平线向右延伸16像素）
- 剑刃高光（半透明白线）
- 剑尖亮点

## 八、渲染调度逻辑

### 怪物渲染（`renderRoom` 中）
```javascript
const circ = scene.add.graphics()
circ.setPosition(x, y)
let drawer = null
const nm = monName.toLowerCase()

if (isFlameSlime) {
  drawer = scene.drawFlameSlime
} else if (/* 哥布林匹配 */) {
  drawer = scene.drawGoblin
} else if (/* 史莱姆匹配 */) {
  // 注意：先检查火焰史莱姆，再检查普通史莱姆
  drawer = scene.drawSlime
} else if (/* 骷髅匹配 */) {
  drawer = scene.drawSkeleton
}
// ... 其他怪物 ...
if (!drawer) drawer = scene.drawGoblin  // 默认哥布林
if (drawer) drawer(circ, 1.5)  // 统一1.5倍缩放，与骑士尺寸一致
circ.setDepth(20)
```

### 奇遇NPC渲染
```javascript
const encounterDrawerMap = {
  CHEST: scene.drawChest,
  MAIDEN: scene.drawMaiden,
  ANGEL: scene.drawAngel,
  BLACKSMITH: scene.drawBlacksmith
}
const npc = scene.add.graphics()
npc.setPosition(npcX, npcY)
const drawer = encounterDrawerMap[event.type]
if (drawer) drawer(npc, 1.5)  // 奇遇NPC使用1.5倍缩放（统一尺寸）
npc.setDepth(20)
```

### 商人渲染
```javascript
const npc = scene.add.graphics()
npc.setPosition(rectCenterX, rectCenterY)
if (scene.drawShopMerchant) scene.drawShopMerchant(npc, 1.5)  // 统一1.5倍缩放
npc.setDepth(20)
```

## 九、关键设计原则

1. **局部原点**：所有绘制以(0,0)为原点，通过`setPosition`放置
2. **向下为Y正**：Phaser坐标系中Y正方向向下
3. **缩放兼容**：所有坐标乘以`s`，支持统一缩放（当前统一使用1.5）
4. **半透明层次**：使用`fillStyle(color, alpha)`实现发光、光环、透明效果
5. **层次覆盖**：先绘制的在底层，后绘制的覆盖上层
6. **尺寸参考**（基础尺寸，渲染时乘以1.5）：普通怪物≈半径10，精英≈半径12，Boss≈半径14-16，NPC≈半径8-10
7. **标签偏移**：实体放大后，底部名称标签Y偏移从24→34，NPC标签从20→30
