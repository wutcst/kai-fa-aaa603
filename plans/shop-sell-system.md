### 前端（1个文件）

| 文件 | 改动 |
|---|---|
| [`GameCanvas.vue`](frontend/src/components/GameCanvas.vue) | 新增 `showShopSell()` 函数（显示背包物品+售价），售卖按钮点击后调用 |

## 数据流

```
用户点击"售卖" → showShopSell() 显示背包物品
    ↓ 用户点击某物品
fetch POST /api/command { command: "shop sell 生命浆果" }
    ↓ 
ShopCommand.executeSell()
    → 检查是否在商店房间
    → 从背包查找物品
    → 如果已装备先卸下
    → 从背包移除
    → player.getMoney().add(售价)
    → 返回更新数据
    ↓
前端刷新显示
```

## 前端售卖界面设计

- 覆盖层显示背包中的所有物品（每个物品一行：图标+名称+售价+售卖按钮）
- 顶部标题"售卖 — 点击物品出售"
- 底部"结束售卖"按钮关闭
